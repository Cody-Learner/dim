# dim<br>

ddcutil wrapper script to control Acer monitor brightness on desktops.		<br>
										<br>
Depends: ddcutil								<br>
										<br>
Usage: dim [-1 -3 -5 -7 -9]							<br>
  -1  Dim	Monitor								<br>
  -3  D/M	Monitor								<br>
  -5  Mid	Monitor								<br>
  -7  M/B	Monitor								<br>
  -9  Bright	Monitor								<br>
